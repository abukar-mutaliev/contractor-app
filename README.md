- Поиск объектов по названию
- Добавление и удаление объектов
- Добавление и изменение отчетов
- Добавление статусов

## Запуск проекта

Для запуска проекта вам необходимо набрать команду в терминале:

```javascript
npm i
```

После набрать команду:

```javascript
npm run dev
```
